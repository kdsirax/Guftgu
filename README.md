# Guftgu - Realtime Private Chat Application

Guftgu is a real-time private chat web application built using React and Firebase. It provides users with a secure authentication, and seamless chatting experience where they can communicate with their friends in real-time.




## Features

- User authentication
- Private chat rooms
- Real-time updates of chat messages
- Sending text messages and images


## Technologies Used

- React
- Firebase Realtime Database
- Firebase Authentication
- Sass
